Build project with CMake

project structure

project

--apps

----inc

------headers.h

----src

------MainFile.cpp

----CMakeLists.txt

----------- cmake_minimum_required(VERSION 3.9.0)
----------- project(app)
----------- add_executable(app
----------- src/MainFile.cpp)
----------- target_include_directories(app PRIVATE inc/)

--build



cmake -B build ./apps

cmake --build ./build

./build/app



check your program file inside build folder
